# Template Literals
As templates literals são formas de incluir variáveis dentro de strings no es6 de forma mais simplificada e fácil.

Ex:
````javascript
const nome = 'Hudson';
const idade = 19;

// Sintaxe Antiga
console.log('Meu nome é ' + nome + ' e tenho ' + idade + ' anos');
// Com ES6
console.log(`Meu nome é ${nome} e tenho ${idade} anos`);
````
